Project Overview

This project demonstrates a comprehensive IPC Framework using: Pipes, Message Queues & Shared Memory.

It helps show how processes communicate efficiently while protecting Confidentiality, Integrity & Availability.

IPC Methods Used

Pipes

Simple and fast unidirectional channel for related processes.

Message Queues

FIFO message passing with priority support.

Shared Memory

Fastest communication for frequent read/write operations.

Framework Flow (Conceptual)

A simplified architecture of IPC mechanisms between processes.
